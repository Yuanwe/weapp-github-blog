# 微信小程序－github博客

### 说明：

实现浏览github博客功能，特色：
- 显示html内容
- 显示markdown内容

### 数据接口:

- https://api.getweapp.com/vendor/github/blog

### 目录结构：

- images — 存放项目图片文件
- pages — 存放项目页面文件
- util — 存放es6 polyfill,marked库文件和日期处理文件
- wxParse — 存放html显示库文件

### 开发环境：

微信web开发者工具 v0.11.112301
